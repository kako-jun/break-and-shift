# 使い方

## 基本的な読み進め方

このサイトは順番に読む必要はないが、章の順序には意図がある。

1. **序章（ホーム）**から始めると、確率と割合の違いという中心テーマをつかめる
2. **第一章〜第五章**は、テーマを段階的に掘り下げていく本編
3. **特別編**は独立したシミュレーターが中心で、どこからでも入れる

---

## 各ページの構造

すべてのページは同じ構造で作られている。

1. **モノローグ**：境界ユウが問いかける。「なぜこれが面白いのか」の導入
2. **シミュレーター**：自分で操作して体験する
3. **統計表示**：操作した結果が数値で見える
4. **考察**：境界ユウが結論を語る
5. **解説**：なぜそうなるのかの理屈をまとめたリスト

---

## シミュレーターの操作

### スライダーが付いているページ

スライダーを動かすと、リアルタイムで計算結果が変わる。

- 第一章：努力レベルを変えても、割合は変わらないことを確かめる
- 第二章：祈りの強さを変えても、確率への影響がゼロであることを見る
- 第四章：天井の回数や確率倍率を動かして、ガチャの仕組みを理解する
- 雨の中を走るか歩くか：距離・雨の強さ・体の大きさを調整してグラフを観察する

### ボタンで試行するページ

- **少数回**（1回・10回）で始めて、偏りが大きいうちの様子を見る
- **多数回**（1000回・10000回）に進んで、確率が収束していく過程を見る
- **リセット**して、もう一度最初から試せる

### Canvas（2D描画）を使うページ

スロット・じゃんけん・クレーンゲームは、ブラウザのCanvas要素で描画されている。

- スロット：「種別開示モード」をオンにしてからスピンすると、演出が始まる前に結果が表示される
- じゃんけん：遅延を50ms以下に設定すると後出しが見えない速度になる。「不正を可視化する」で検出できる
- クレーンゲーム：アームの動きをアニメーションで見ながら、握力の変化を数値で追う

### 3D（Three.js）を使うページ

チンチロリン（サイコロ）ページには3Dシーンが含まれる。

- マウスドラッグでカメラを回転できる
- スクロールで拡大縮小できる
- 3Dシーンはあくまで演出。統計は画面下部の数値で確認する

## グラフの読み方

### 折れ線グラフ（雨の中を走るか歩くか、先祖の確率）

- 横軸：時間の経過または世代数
- 縦軸：濡れる量または先祖の数
- 複数の線が同じグラフに描かれているときは、線が離れている部分と交差する部分に注目する

### 棒グラフ（生存者バイアス）

- 「突然死した恒星を推定表示」チェックボックスをオフ→オンにすることで、見えているデータと見えていないデータの差に気づける

---

## 統計的な理解を深めるために

以下の順序で体験すると効果的。

1. **少ない試行回数**で動かして、偏りが大きい様子を見る
2. **試行回数を増やす**にしたがって、偏りが小さくなっていくことを確認する
3. **考察**のモノローグを読んで、何を意味するのかを整理する

カイ二乗検定（チンチロリンページ）では、試行回数が100回以下だと「偏りが検出された」と表示されることが多い。これは「サイコロが歪んでいる」のではなく、「まだ試行が少ないため偶然の偏りが統計的に有意に見える」という意味だ。試行を増やすと次第に「偏りなし」に収束する。

## サイドバーのナビゲーション

画面左側（大きなスクリーン）または上部メニュー（小さなスクリーン）から、各章へ移動できる。

- 序章（ホーム）
- 第一章〜第五章（本編）
- 特別編：チンチロリン（3D物理演算）

特別編のその他のシミュレーター（スロット、じゃんけん、クレーンゲーム、雨歩行、生存者バイアス、前世確率）は、ホームページの一覧カードからアクセスできる。
